# DuckDash

**DuckDash** is a virtual sandbox mainly inspired by Club Penguin. Explore, chat with friends, and waddle around as a duck!

[Live Demo](https://duckdash.github.io)

---

## Features

* **Custom Flash-Style Player:** You want Flash? You don't? How 'bout HTML5? No? An iFrame???? :)
* **Live Chat:** Send messages that appear in stylized speech bubbles above your duck.
* **Classic UI:** UI that takes you straight back to Club Penguin! Wowza!

---

* **HTML5 & CSS3:** You want the actual website? Here, take Hypertext Markup Protocol and Cascading Style Sheets. Too much words for you? TOO BAD!
* **JavaScript:** You want the game to work? Here, take JavaScript.
* **Node.js:** For da servers

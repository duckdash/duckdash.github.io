# DuckDash

**DuckDash** is a virtual sandbox mainly inspired by Club Penguin. Explore, chat with friends, and waddle around as a duck!

[Live Demo](https://duckdash.github.io)

---

## Features

* **Custom Flash-Style Player:** A sleek, borderless 1080x720 (scaled) game window with 17px rounded corners.
* **Constant-Speed Movement:** Authentic walking mechanicsâ€”no matter the distance, your duck moves at a steady pace.
